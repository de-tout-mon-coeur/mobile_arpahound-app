Arpahound - мобильное приложение для анализа трафика (Wireshark), проверки скорости интернета (Speedtest) и надежности подключения (Ping latency, loss).
Серверная часть написана на С, использует библиотеку libpcap для захвата пакетов и сетевую библиотеку mongoose.
